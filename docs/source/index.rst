Welcome to KoolProxies documentation!
===================================

**KoolProxies** is a proxy service to find reliable proxies in seconds.
It was created by [Kool Services](https://kool.services) (feel free to check out our other services).

Check out the :doc:`usage` section for further information, including
how to use the API.

.. note::
